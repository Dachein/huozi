# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). The full text is canonical; this file is a short pointer plus the project-specific bits.

## Short version

- Be civil. Assume good faith.
- Focus on the work — disagree about ideas, not people.
- Avoid harassment, discriminatory jokes, personal attacks, unwelcome sexual attention, doxxing, or any of the obvious bad-faith stuff.
- Respect the maintainers' time. Help us help you by writing tight, reproducible bug reports.

## Where it applies

The whole project space: GitHub issues, PR comments, commit messages, discussions, the project's email and other contact channels listed in [SECURITY.md](SECURITY.md). It also applies to representations of the project in public — talks, blog posts, social media — when you're identifying as a contributor.

## Reporting violations

Email `dachein.x@gmail.com` with subject prefix `[huozi-conduct]`. Reports are confidential. We'll acknowledge within 3 business days.

For security issues (separate concern), use [SECURITY.md](SECURITY.md) instead.

## Enforcement

Maintainers will respond proportionally:

1. **Private clarification** — for first-time slips that look like miscommunication.
2. **Public correction** — when a public statement was harmful and clarity matters.
3. **Temporary block** — for repeated or severe violations.
4. **Permanent ban** — for sustained pattern, retaliation against reporters, or any single instance of harassment / threats.

The maintainers' decision is final, but you can appeal once with new context.
