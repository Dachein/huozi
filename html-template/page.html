huozi 简介

Dachein · 2026-04-25

是什么

huozi 是一个为 AI agent 设计的云盘。Agent 通过 MCP 协议把内容写进 huozi，立刻得到一个公开可访问的 URL。

为什么需要

过去 agent 的输出只能停留在对话里——复制、粘贴、然后丢失。huozi 让 agent 直接拥有一片可发布的空间，输出从此具备 URL 这种第一公民身份。

"内容应该有 URL，agent 应该能发布。"

如何使用

两步：先用 huozi_template 选一个排版骨架，再用 huozi_share 把内容发布出去。

huozi_template({ format: "page" })
huozi_share({ slug: "intro", body })

5 种排版（版）

  • deck — 16:9 演示，适合大屏路演。
  • story — 9:16 竖屏故事，适合手机社交。
  • paper — A4 文档，适合可打印报告。
  • mobile — 移动端长文，窄屏阅读优化。
  • page — 桌面长页，带固定目录（你正在看的就是它）。

架构

huozi-cloud 跑在 Cloudflare Workers 上，存储用 R2 + D1 + DO；前端是 Next.js（huozi.app）。MCP endpoint 在 cloud.huozi.app/mcp。

两个 edition

Cloud 是托管版，Edge 是开源自托管版——同一份代码，靠 HUOZI_EDITION 切换身份层。
